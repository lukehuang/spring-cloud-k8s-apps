Spring Boot fronend-app running on k8s

This application is using Spring Cloud Kubernetes to connect a frontend app and a backend app.

Spring boot based app container images are built using Cloud Native Buildpacks (look Ma, no Dockerfile!):

$ pack build myrepo/myimage

The backend-app Spring Boot container can be scaled using k8, and see client-side load balancing & service discovery in action:

$ kubectl -n spring-cloud-k8s scale --replicas=2 deployment backend

Edit ConfigMap objects and see how the new configuration is applied:

$ kubectl -n spring-cloud-k8s edit configmap backend
